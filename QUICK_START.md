# Быстрый старт HTTPS

## Запуск
```bash
npm run dev-https
```

## Доступ
```
https://localhost:443
```

## Остановка
```bash
pkill -f "node server.js"
```

## Проверка
```bash
curl -k https://localhost:443
```

---
**Примечание**: При первом посещении браузер покажет предупреждение о небезопасном соединении. Нажмите "Дополнительно" → "Перейти на localhost (небезопасно)".
